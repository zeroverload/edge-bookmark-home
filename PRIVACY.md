# Privacy Policy

Bookmark Home is a local-first browser extension for displaying and organizing browser bookmarks on the new tab page.

## Data Access

The extension requests these permissions:

- `bookmarks`: reads bookmark titles, URLs, folders, and can move bookmarks when you drag a bookmark to another folder card in edit mode.
- `storage`: stores your local display settings, including home title, card order, card display names, bookmark aliases, and tags.
- `favicon`: displays website icons next to bookmarks.

## Data Storage

Bookmark Home stores settings only in the browser's local extension storage via `chrome.storage.local`.

Stored settings may include:

- Custom home title.
- Card display order.
- Card display names.
- Bookmark aliases.
- Bookmark tags.
- Last selected read/edit mode.

## Data Sharing

Bookmark Home does not collect, upload, sell, share, or transmit your bookmark data or settings to any server.

The project does not include analytics, telemetry, advertising SDKs, remote scripts, or third-party tracking.

## Bookmark Changes

In read-only mode, the extension does not modify your browser bookmarks.

In edit mode, dragging a bookmark to another folder card uses the browser bookmarks API to move that bookmark in your real browser bookmarks.
