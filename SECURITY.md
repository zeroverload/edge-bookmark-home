# Security Policy

## Reporting a Vulnerability

If you find a security issue, please open a private security advisory on GitHub if available, or contact the maintainer listed in the repository.

Please include:

- A clear description of the issue.
- Steps to reproduce it.
- Expected and actual behavior.
- Browser and extension version.

## Security Notes

Bookmark Home is designed to run locally in the browser. It does not use remote code, analytics, telemetry, or network requests.

The extension uses the `bookmarks` permission. In edit mode, moving a bookmark between cards changes the bookmark's real folder in the browser.
