# Changelog

## 0.1.0

- Initial MVP release.
- Replaces the new tab page with a bookmark dashboard.
- Displays bookmark folders as cards.
- Supports bookmark search by title, URL, folder path, alias, and tags.
- Supports read/edit mode.
- Supports custom home title, card display names, card ordering, bookmark aliases, and tags.
- Supports moving bookmarks between folder cards in edit mode.
